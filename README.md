# TradeMark Evolution

J-PlatPatから出力した商標CSVデータを、ブラウザ上で分析・可視化するツールです。

商標（TradeMark）の情報を活用し、意思決定の質を「進化」させたいという思いから、**TradeMark Evolution**と名付けました。

## 主な機能

- 商標CSVデータの読み込み
- 出願人・商標区分等の分析
- グラフによる可視化
- 分析結果の確認・出力

※搭載されている機能は、公開バージョンによって異なる場合があります。

## 使用方法

### ブラウザ上で利用する

[TradeMark Evolutionを起動する](https://estoppel88.github.io/trademark-evolution/)

上記リンクから、ダウンロード不要で利用できます。

### HTMLファイルをダウンロードして利用する

1. [`260719_TradeMark_Evolution_v1.0.html`](260719_TradeMark_Evolution_v1.0.html)をダウンロードします。
2. ダウンロードしたHTMLファイルを、Google ChromeやMicrosoft Edgeなどのブラウザで開きます。
3. 画面の案内に従って、分析対象のCSVファイルを読み込みます。
4. 集計条件や表示条件を設定し、分析結果を確認します。
インストール作業は必要ありません。

## 動作環境

最新版の以下のブラウザでの利用を推奨します。

- Google Chrome
- Microsoft Edge

ブラウザの設定やバージョンによっては、正常に動作しない場合があります。

## データの取り扱い

本ツールに読み込ませたCSVデータおよび分析結果は、お使いのブラウザ内で処理されます。
本ツールには、読み込んだCSVデータを作成者やGitHubなどの外部サーバーへアップロードする処理は設けていません。
ただし、グラフ表示などに必要な外部ライブラリをインターネット経由で読み込むため、ツールの起動時などにインターネット接続を使用します。

機密情報や個人情報を含むデータを利用する場合は、所属組織の情報管理ルールをご確認のうえ、利用者自身の責任でご使用ください。
